# CloudWatch Monitoring

## Objective
The objective of this step was to monitor the health of the EC2 instance using AWS CloudWatch metrics.

## Metrics Observed
- CPUUtilization
- StatusCheckFailed

These metrics were used to understand instance performance and availability.

## Monitoring Setup
Default CloudWatch monitoring was enabled for the EC2 instance.
Key metrics were continuously observed to identify abnormal behavior and performance issues.

## Outcome
EC2 instance metrics were successfully monitored using CloudWatch, providing visibility into system health.
